### 我的个人博客站

#### 步骤：

1. 拉取

```bash
git pull git@github.com:liruoyu2008/blog.git
```
2. 进入

```bash
cd ~/blog 
```

3. 安装nodejs

```bash
apt install nodejs
```

4. 安装hexo

```bash
npm install hexo-cli -g
```

5. 安装modules

```bash
npm install
```

6. 清理

```bash
hexo clean
```

7. 生成pages

```bash
hexo g
```

8. 部署到本地

```bash
hexo s
```

9. 部署到Github Pages（需要在_config.yam进行相关配置）

```bash
hexo d
```
